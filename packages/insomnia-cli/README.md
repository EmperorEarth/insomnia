# insomnia-cli

### Usage
```
npm install -g insomnia-cli
inso --version
```

### Development
- Bootstrap: `npm run bootstrap`
- Start the compiler in watch mode: `npm run watch`
- Run: `node bin/inso -v`

> You can also set `bin/inso` as an executable (`chmod +x bin/inso`) and run as `./bin/inso -v`
